pub struct User {
    pub username: String,
    pub groups: Vec<u64, Global>,
    pub roles: Vec<u64, Global>,
    pub password_hash: Option<ServerFile>,
    pub pending_password_change_state: Option<ServerRegistration>,
}

A user that can authenticate with PliantDb.

Fields

username: String

The name of the role. Must be unique.

groups: Vec<u64, Global>

The IDs of the user groups this user belongs to.

roles: Vec<u64, Global>

The IDs of the roles this user has been assigned.

password_hash: Option<ServerFile>

An OPAQUE PAKE payload.

pending_password_change_state: Option<ServerRegistration>

A temporary password state. Each call to SetPassword will overwrite the previous state.
